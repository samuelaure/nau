## Tasks

### 6.1 Recursive Journal Engine
- [x] Create `src/journal/journal.module.ts`.
- [x] Refactor `JournalService` to support **Recursive Context Aggregation**:
  - When summarizing a period (e.g. Weekly), first fetch existing `journal_summary` blocks within that range.
